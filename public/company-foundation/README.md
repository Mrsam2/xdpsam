# XDP technologies — Company Foundation

> **XDP's = eXplore • Develop • Prosper**

This repository is the single source of truth for how **XDP technologies** operates. It contains our company identity, governance, policies, and the playbooks that run every function of the business. If a process is not written down here, it is not official.

**Official website:** [xdptechnologies.dev](https://xdptechnologies.dev)

## Why this repository exists

We treat the company itself as a product. Like code, our processes should be versioned, reviewed, and improved through merge requests. Anyone proposing a change to how we work opens an MR against the relevant document.

## Brand at a glance

- **Vision:** Build technology that empowers people, businesses, and industries worldwide.
- **Mission:** Explore innovation. Develop impactful solutions. Prosper together.
- **Tagline:** Explore. Develop. Prosper.

> **Note on the name:** "XDP's" is also a known Linux networking acronym (eXpress Data Path). We consistently brand ourselves as **"XDP technologies"** to build a distinct identity.

## Contributing to these docs

1. Create a branch.
2. Edit or add the relevant Markdown file.
3. Open a merge request and request review from a Co-founder.
4. Merge after approval. Changes are effective once merged to `main`.

See [`governance/decision-making.md`](governance/decision-making.md) for how decisions are approved.
